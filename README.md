## Description

Programming test

API to get Github repos by language

Only one endpoint: /git/:lang?page=number

Due to github limitation, only 1000 results can be acquired by search.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
